
1. Validación de correo electrónico:

Objetivo: Asegurar que el usuario ingrese un correo electrónico con un formato válido.

Pasos:
-Validar la longitud: El correo electrónico debe tener una longitud mínima y máxima.
-Validar el formato: Debe contener un símbolo "@", un nombre de dominio y una extensión.


2. Validación de contraseña:

Objetivo: Asegurar que el usuario ingrese una contraseña segura.

Pasos:
-Validar la longitud: La contraseña debe tener una longitud mínima de 8 caracteres.
-Validar la complejidad: Debe contener al menos una letra mayúscula, una letra minúscula, un número y un símbolo especial.


3. Validación de ID:

Objetivo: Validar un ID.

Pasos:
-Validar que sean 2 numeros y 5 letras.


4. Validación de un nombre:

Objetivo: Asegurar que el usuario ingrese un nombre correcto.

Pasos:

-Validar que ingrese solo letras, espacios y puntos; máximo 40 caracteres


5. Validación de una clave:

Objetivo: Asegurar que el usuario ingrese una dirección correcta.

Pasos:

-Valida que ingrese mínimo 3 letras sin caracteres especiales ni números; no es obligatorio

1. Validación de número de teléfono:

Objetivo: Asegurar que el usuario ingrese un número de teléfono con un formato válido.

Pasos:
-Validar la longitud: El número de teléfono debe tener 10 digitos.
-Validar el formato: Debe contener solo números.


2. Validación de un código postal:

Objetivo: Asegurar que el usuario ingrese un código postal válido.

Pasos:

-Validar la longitud: El código postal debe tener 5 digitos.
-Validar el formato: Debe contener solo números.


3. Validación de una dirección:

Objetivo: Asegurar que el usuario ingrese una dirección correcta.

Pasos:

-Validar la longitud: El código postal debe tener 5 digitos.
-Validar el formato: Debe contener solo números.


4. Validación de un apellido:

Objetivo: Asegurar que el usuario ingrese un apellido correcto.

Pasos:

-Validar que ingrese solo letras y espacios; máximo 20 caracteres


5. Validación de una descripción:

Objetivo: Asegurar que el usuario ingrese una descripción.

Pasos:

-Validar que ingrese cualquier caracter excepto @, ~, `; no es obligatorio.
